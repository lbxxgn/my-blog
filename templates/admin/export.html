导出选项

选择导出格式，导出的文件将保存在服务器的 exports/ 目录中。

📝

Markdown 格式

将每篇文章导出为独立的 Markdown 文件，包含完整的元数据（标题、日期、分类、标签）

  • ✓ 便于版本控制
  • ✓ 支持静态站点生成器
  • ✓ 人类可读格式
  • ✓ 包含完整文章内容
导出 Markdown
📦

JSON 格式

将所有文章导出为一个 JSON 文件，包含完整的数据库结构和内容

  • ✓ 完整的数据备份
  • ✓ 易于程序处理
  • ✓ 支持数据迁移
  • ✓ 包含所有元数据
导出 JSON

📌 导出说明

  • 导出的文件保存在服务器 exports/ 目录中
  • 每次导出都会创建新的文件，不会覆盖之前的导出
  • Markdown 文件命名格式：ID_Title.md
  • JSON 文件包含时间戳：blog_export_YYYYMMDD_HHMMSS.json
  • 请定期下载导出文件到本地进行备份

💻 命令行导出

你也可以通过命令行直接导出：

cd /Users/gn/simple-blog
python3 backend/export.py
