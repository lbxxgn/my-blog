# 2014年01月14日

&nbsp; &nbsp; &nbsp; &nbsp; 1、回想起去年考研之后的心情，那真是世界末日走投无路的感觉，那时的我考完感觉一切都完了，我需要再考一年，怎么面对家人，怎样继续接下来的生活，怎样面对周围的同学。<div>&nbsp; &nbsp; &nbsp; &nbsp; 2、第二次是考研调剂的时候那种不知所措，那种大脑一片空白，那种完全不能理智思维的状态，哎~~~想跟人说却又不知道该和谁说。那样的无助与堕落，那时的自己感觉毫无尊严，毫无生活的快乐。</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 3、第三次是这回期末考试，考的那时惨不忍睹，那时一个郁闷，那是一个自卑，那种凄凉，世界末日的感觉。现在要好好的思考一下，不能再来第四次，第五次的这种走投无路的感觉。自己需要努力了，需要认真起来，不能再每天浑浑噩噩的、犹如生活在幻想之中，要走入生活，发现生活的美好~~~</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 4、要认真起来，不能再凑合着每天的生活，要关注生活中的小事、琐事，不要再去想一些与自己无关的大问题，大方向了，要注重自己的小生活了~~~</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 5、最近一段时间的自己是那么的消沉，要让生活充满阳光，要无私的面对生活，要敞开心扉，面对自然，面对生活~~~</div>
