# 2014年02月07日

&nbsp; &nbsp; &nbsp; &nbsp; 今天是2月7号，大年初八，距离真正开学时间2月24号还有17天，但是我这么早的来到了学校，这是我来学校最早的一天，研究生生活的第二个学期开始了。总结一下这个假期，这个假期我总共在家待了不到15天，在家里爸爸妈妈对我是那样的好，不知为什么现在爸爸妈妈也不会叫我去做这做那了，他们只会默默地做着一切，看着爸爸妈妈一天天的老去，自己不知道该说些什么，也不知道该做些什么，只是心里真的很难过。家里买了新电视，昨天一天早上懒洋洋的睡起来，妈妈帮我放好水，我就去洗了个澡，洗完澡就和妈妈去取了我的火车票，然后去菜市场买了些五花肉，买了点菜。回到家里爸爸开始做饭，下午一家三口边看电视边睡了一会。想一想自己什么时候才能有个女朋友呢~什么时候才能有自己的另外一个小家呢~~？看着爸爸妈妈的同事朋友的孩子都有了自己的另一半，可是我呢？什么时候可以有一个让人心仪的女朋友呢~~~自己也24岁了，有的同学到了这个年龄都已经结婚有孩子了，可是我呢~~~哎 ~·<div>&nbsp; &nbsp; &nbsp;自己的眼界也要放宽一些，要做一些自己想做的事情。要靠自己养活自己了，要学习一些真正有用的东西了，要多和别人交流了~~~~</div><br><div>这个学期的时间需要规划一下，自己真的要有一定的节奏了，不能再像上个学期那样的混乱了</div><div>1、最重要的是学好3门学位课</div><div>2、认真准备6级，一定要背单词、练习听力</div><div>3、自己要学一些操作系统啊，数据结构啊~~学一些有用的技能与知识</div><div>4、处理好实验室这面的东西，分配好时间，不能将大把的时间浪费给实验室</div><div>5、看一些自己喜欢的课外书，提高自己的见识</div><div><br></div><div>要放开自己的眼界，不能仅仅将目光放在生活在自己周边的人身上，不能仅仅局限于学校，自己现在不仅仅是一个学生，这仅仅是为我以后的一种手段·~~~</div><div><br></div><div>★最最重要的是什么时候才可以找到自己的另一半啊~~~~</div><br><div>话说我自己的目标是什么呢？是赚钱吗？我想不是的。如果是纯粹的赚钱，那么我会感到生活很累，那么我想要的是不需太多的钱，能够过上不是那么累的生活，生活的安稳与踏实~~~那么自己需要做什么才能达到这样的目标呢~~~</div><br><div>自己~~~</div>
