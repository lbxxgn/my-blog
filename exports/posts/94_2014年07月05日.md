# 2014年07月05日

&nbsp; &nbsp; &nbsp; &nbsp; 时间过得好快啊~~~研一就这样过去了~~~最近的心情非常不好，总是有考试，我现在是畏惧了这里的种种的考试，结果是我感觉我复习的还算可以啊~~但是我依旧挂了，不知道下个学期可不可以重修~~~上课真的没什么用啊~~<div><br></div><div>&nbsp; &nbsp; &nbsp; &nbsp; 我发现每次我写博客都是在感慨时间过的好快~~~真的，这一年来自己的所有计划都没有实现，匆匆的一年就不知不觉的过去了，每天的自己就是睡觉，玩手机，玩电脑，刷博客，被各种杂乱零碎的消息占去了我的大把时间。</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 本来是想学习一下编程知识的，可是现在呢~~~没时间了，还有一年半的时间，具体来说就是一年的时间了，最后一个学期要写毕业论文和找工作。现在的我都后悔为什么要来读研了~~~</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 读研，但是放假连家都回不了，消耗着和工作一样的时间，但是没有一点的回报，要通过这种时间来学习。本来说上演的时候要多看看书的，但是这个学期以来自己都很少的去图书馆，更何谈读书了，每天的自己就是一觉睡到很晚，起来玩上两个小时，然后去吃饭，然后回来继续睡觉，下午去实验室坐在那里继续玩，晚上也是一直都在玩，总是控制不住自己想去动动手机。上研了也没有认识几个人，每次的考试都把我搞得神魂颠倒，心惊肉跳，六神无主，最终还是不能得到好的成绩。</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 自己的自卑，和同学亲人间的联系是少之又少，现在的自己真是形单影只。自己缺乏胆量与眼识，不能很好地判断方向，好多东西都是自己说来要坚持去做的，但是确是说过之后转眼就忘记而没有去做的，更何谈坚持？？</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 从现在开始，要充分地利用自己的时间，不能再随意的浪费了，要做一个有心人，要做一个善于思考的人~~~做一个善于行动的人。</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 自己的思想总是潮起潮落，这几天在想这个事，可是过几天就把之前的事给忘了，去想另一件事了，真是狗熊掰棒子，掰一个扔一个，最后什么都没有得到，都是无用功~~~</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 自己总是处在压抑之中，心情非常的不好，不善于与别人沟通。每天的生活没有朝气，我一定要改变。外界的因素不能困扰我，要比较好的处理外界因素的影响，从心出发接触外界。找工作，去哪里？找什么样的工作？</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;上天啊~~~不过我相信我一定会好转的，自我治疗~~~~</div>
