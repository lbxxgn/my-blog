# 承诺书

<div>
&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 亲爱的同学，欢迎你走进西安电子科技大学！为了让大家更好地了解我校研究生学风建设的相关规定，督</div><div>&nbsp; &nbsp; 促研究生遵守学习纪律，营造诚实守信的学习风气。特制作此承诺书，请大家认真阅读后签名承诺，并自觉</div><div>&nbsp; &nbsp; 在实践中践行。 &nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 一、在校期间遵守宪法、法律、法规和学校的各项管理制度；勤奋学习、努力实践，完成规定学业；养成</div><div>&nbsp; &nbsp; 良好的思想品德和行为习惯；按规定缴纳培养费及有关费用。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 二、自觉维护校园秩序，营造稳定安全的学习和生活环境。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 三、严格遵守请销假制度。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 四、按时参加培养计划规定的教学和科研活动。自觉遵守学习纪律，上课不迟到、不早退、不旷课，课堂</div><div>&nbsp; &nbsp; 认真听讲，不得使用移动电话，上课时间不得开展与学习无关的活动。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 五、自觉维护教学秩序，遵守学校规定，不得在教学、科研场所从事与学习、科研无关的活动。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 六、自觉培养科研精神，提高创新能力，养成良好的学术道德习惯，坚决杜绝抄袭、剽窃他人学术成果。</div><div>&nbsp; &nbsp; 认真学习并贯彻《西安电子科技大学研究生学术行为规范条例（试行）》的相关规定。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 七、自觉遵守考试纪律，不违规不作弊。认真学习并贯彻《西安电子科技大学研究生管理规定（试行）》</div><div>&nbsp; &nbsp; （西电学〔2010〕24号）、《西安电子科技大学研究生管理实施细则》、《西安电子科技大学学生违纪处分办</div><div>&nbsp; &nbsp; 法（试行）》（西电学〔2010〕43号）中的相关规定。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 八、在学习活动中，树立良好学习风气，勤奋学习、刻苦钻研、自强不息追求真理。认真阅读并自觉遵守</div><div>&nbsp; &nbsp; 《西安电子科技大学研究生学风诚信档案管理暂行办法》中的有关规定。</div><div>&nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 上述各项规定由各学院负责组织研究生学习。</div><div>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</div><div>&nbsp; &nbsp; &nbsp; &nbsp; 承 &nbsp;诺：本人已认真学习并熟知上述的各项规定，在校期间一定认真遵照执行，诚实守信，严于律己，如</div><div>&nbsp; &nbsp; 有违反，愿意接受处罚。 &nbsp;&nbsp;</div><div>
