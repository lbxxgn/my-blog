# Passkey 操作手册

## 这是什么

Passkey 是网站的快捷登录方式。

在这个系统里：

- `iPhone` 上通常会调起 `Face ID`
- `Mac` 上通常会调起 `Touch ID`

你仍然可以继续使用用户名密码登录。

---

## 日常使用

### 第一次启用

1. 用账号密码正常登录
2. 进入 `/change-password`
3. 找到“快捷登录”区域
4. 点击 `为当前设备启用快捷登录`
5. 按系统提示完成 Face ID / Touch ID 验证

启用成功后，这台设备就绑定好了。

### 后续登录

1. 打开 `/login`
2. 点击 `使用 Face ID / Touch ID 登录`
3. 按系统提示完成验证
4. 成功后自动进入后台

---

## 多设备建议

如果你主要在 `iPhone + Mac` 上使用，建议这样做：

1. 在 Mac Safari 上绑定一次
2. 在 iPhone Safari 上再绑定一次
3. 在“快捷登录”列表里给设备起清晰名称

例如：

- `我的 MacBook`
- `我的 iPhone`

---

## 设备管理

在 `/change-password` 页面可以看到已绑定设备列表。

支持操作：

- 查看已绑定设备
- 删除旧设备

适合这些情况：

- 设备更换
- 不再使用旧手机
- 想重新绑定

---

## 常见问题

### 1. 为什么没有直接弹 Touch ID / Face ID

通常是因为这台设备还没先绑定过 Passkey。

请先：

1. 密码登录
2. 去“快捷登录”里绑定当前设备

### 2. 为什么会看到“iPhone、iPad 或安卓设备 / 安全密钥”

这通常说明浏览器没有找到本机已绑定的 Passkey，于是退回到跨设备登录或外部安全密钥模式。

### 3. 为什么报域名无效

本地开发时请尽量使用：

```txt
http://localhost:5005
```

不要直接用：

```txt
http://127.0.0.1:5005
```

### 4. 忘了密码怎么办

Passkey 只是快捷登录，不是密码找回方案。

如果你还没绑定 Passkey 或设备不可用，仍然需要通过原密码登录。

---

## 推荐使用方式

如果这是你自己的个人系统，我建议：

1. 保留一个你记得住的强密码
2. 在常用的 Mac 和 iPhone 上各绑定一次 Passkey
3. 日常优先用 Passkey
4. 修改安全设置时仍保留密码作为兜底
